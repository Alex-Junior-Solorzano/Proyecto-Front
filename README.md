## Run it Locally


 ### Setup
 
  - Clone this repo using `git clone https://github.com/Alex-Junior-Solorzano/Proyecto-Front`

  - Go the proyect folder, open a terminal and following commands

 ### Comands


  - To install all packages

    `npm i`

  - To run app

    `npm start`
